Library


You can borrow books, DVDs, music, and many things from the library.


You can use a computer for free. You can read books and magazines.


Your children can borrow books. Your children can get help with homework.


You need to have a library card. The library card is free.


You need to return books on time. You need to pay money if you return things late or if things are lost.


How many things can I borrow at a time?

You can borrow up to 100 things for each library card. You can borrow:

  • Audiobooks on CD: 25

  • Music CDs: 25

  • DVDs: 25

Do I have to return things to the library where I borrowed them?

You can return things to any Saint Paul Public Library. When a library is closed, things can be put in after-hours book returns.

Can I borrow items from another library?

You can borrow things from almost any library in Minnesota and beyond.
